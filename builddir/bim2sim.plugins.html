class bim2sim.plugins.Plugin¶

Bases: object

Base class for bim2sim Plugins.

Notes:

This class is used as a namespace. Instantiation is not necessary.

Attributes:

name: Name of the Plugin default_workflow: default workflow to use in Projects using this Plugin tasks: Set of tasks made available by this Plugin default_tasks: List of tasks, which should be executed elements: Additional Elements made available by this Plugin
